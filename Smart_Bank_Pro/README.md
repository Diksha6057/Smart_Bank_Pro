💳 Smart Bank System 
A C++ based banking management system to create accounts, deposit and withdraw money and manage records using OOP concepts through a menu-driven interface..

Concepts Used:
1.Inheritance
2.Polymorphism
3.Abstract Base Class
4.Exception Handling 

Features :
1.Deposit Money
2.Withdraw Money
3.Check Balance
4.View Transaction History
5.Interest Calculation (Savings Account)
6.Multiple Account Types:
   i.Savings Account
   ii.Current Account
   iii.Loan Account
7.Exception Handling for:
   i.Invalid inputs
   ii.Insufficient balance
8.Menu-driven user interface
